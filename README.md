## 技術的特徴
- **フレームワーク不使用**: 全てのアプリはVanilla JavaScript、HTML、CSSで構築
- **単一ファイル構成**: 各アプリは依存関係のない単一のHTMLファイル
- **レスポンシブデザイン**: モバイル対応
- **多言語対応**: 日本語UIを中心とした設計

## 使用方法
各アプリケーションのHTMLファイルをブラウザで直接開くだけで動作します。ビルドプロセスは不要です。
